INFTY Documentation
===================

INFTY is an optimization-centric toolkit for Continual AI. It provides optimizers, visualization utilities, and analysis tools for continual-learning research.

This Sphinx documentation tree is optional. The main human-written documentation lives in the Markdown files under ``docs/`` and can be rendered with MkDocs. The RST pages here are useful for ReadTheDocs-style generated API references.
